# Web Text Editor

## Project Overview

Web Text Editor is an ASP.NET web-based text editor designed for real-time content creation. This project focuses on bridging the gap between complex backend file management and a responsive web interface, demonstrating full-stack integration principles.

## Key Features

- Real-time text editing capabilities
- Full-stack integration between C# backend and web front-end
- High-performance client-side responsiveness
- Data reliability on the server side
- File handling and persistence

## Technical Stack

- ASP.NET
- C#
- JavaScript
- HTML5/CSS3

## How to Run

1. Ensure .NET and Visual Studio are installed
2. Open the project in Visual Studio
3. Build the solution:

   dotnet build

4. Run the application:

   dotnet run

5. Open a browser and navigate to `https://localhost:5001`
